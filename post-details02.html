Mettre en place un télétravail
  • Lucas
  • 14 Juin, 2021
  • 0 Comments

Pour mettre en place un télétravail à deux, il a fallu s'adapter avec les outils qui sont disponibles pour pouvoir être le plus efficace possible. Nous avons utilisé des outils comme : Discord pour la communication, Trello pour l'organisation des taches, Github pour se partager le code, Vscode qui sert d'IDE pour coder, Laragon pour héberger localement le site web avant de le publier. Avec tout cela nous avons pu être optimum sur notre travail
